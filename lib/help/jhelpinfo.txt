=== 変愚蛮怒オンライン ヘルプ コマンド一覧 ===

q        : オンラインヘルプを終了する。
<, ESC   : 前のヘルプページに戻る。または終了する。
?        : オンラインヘルプの使用法（このファイル）

Home/End               : ファイルの先頭/最後に移動
-/space, PgUp/PgDown   : １ページ戻る/進む
8/2, 矢印キー↑/↓     : 上/下へ１行移動

#      : 指定した行に飛ぶ (デフォルトは 0行)
%      : 指定したファイルを開く (デフォルトは "jhelp.hlp")
=      : 特定文字を含んだ行を強調表示する
/,^S   : 文字列を検索 ( '/' + Return で再検索)
           ( '#' + '0' + Return + '/' + Return で先頭から検索)
|      : 現在のファイル内容を保存する。
[a-z]  : メニューを選択。または、リンクされた別ファイルに移動する。
         本文中に「(jgeneral.txt#First [a]を参照)」等と書かれていると
         き'a'を押すと参照する場所に移動します。
